# M-Pesa Donation Page

Accept donations via M-Pesa STK Push. Zero callbacks needed.

## Setup

1. Get your Daraja credentials (free, 2 minutes):
   - Go to developer.safaricom.co.ke
   - Sign up or log in
   - Go to My Apps, create a new app
   - Copy your Consumer Key and Consumer Secret

2. Set credentials and run:
   ```bash
   export DARAJA_CONSUMER_KEY=your_key
   export DARAJA_CONSUMER_SECRET=your_secret
   npm install
   npm start
   ```

3. Open http://localhost:3000

The app runs in sandbox mode — no real money is charged.
